# Sistema de Auditoria MAHLE

## Sobre o Projeto

Sistema de auditoria desenvolvido para gerenciar máquinas, auditorias, checklists e relatórios.

## Como configurar e usar

### Requisitos
- Node.js & npm instalado - [instalar com nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

### Passos para começar

```sh
# Passo 1: Clone o repositório
git clone <YOUR_GIT_URL>

# Passo 2: Navegue até o diretório do projeto
cd <YOUR_PROJECT_NAME>

# Passo 3: Instale as dependências necessárias
npm i

# Passo 4: Inicie o servidor de desenvolvimento
npm run dev
```

### Editar arquivos
- Edite os arquivos localmente e commit as mudanças
- Os arquivos serão refletidos no sistema após o push

## Tecnologias Utilizadas

Este projeto é construído com:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Funcionalidades

- Cadastro e gestão de máquinas
- Geração de etiquetas com QR Code
- Sistema de checklists e auditorias
- Relatórios e análises
- Cronograma de auditorias
- Auditoria mobile
